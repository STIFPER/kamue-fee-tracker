# Master Prompt

## ก่อนสร้างหน้าเว็บทุกครั้ง ให้อ่าน

- `project.md`
- `designs.md`
- `colors.md`
- `typography.md`
- `features.md`
- `calculations.md`
- `permissions.md`
- `database.md`
- `coding-rules.md`

## กฎการทำงาน

- สร้างระบบให้สอดคล้องกับเอกสารทั้งหมด
- ห้ามเดาสี
- ห้ามเดาฟอนต์
- ห้ามสร้าง UI ใหม่เอง
- ใช้ Design System เดียวกันทั้งเว็บไซต์
- เน้น Responsive
- รองรับมือถือเป็นอันดับแรก
- ทุก Component ต้อง Reusable
- ทุกหน้าใช้ Layout เดียวกัน
- Code ต้องสะอาด แก้ไขง่าย
- ขยายระบบได้ในอนาคต
