# Project Vision

พัฒนาระบบบริหารค่ามือผู้ช่วยแพทย์ที่ใช้งานง่าย ทันสมัย และยืดหยุ่น รองรับการบันทึกงานประจำวัน การคำนวณค่ามืออัตโนมัติ การติดตามรายได้ และการจัดการข้อมูลของทีมทั้งหมดผ่านระบบเดียว

ระบบต้องรองรับการใช้งานทั้งบนโทรศัพท์ แท็บเล็ต และคอมพิวเตอร์ โดยให้ประสบการณ์ใช้งานที่รวดเร็ว เรียบง่าย และลดขั้นตอนการทำงานให้มากที่สุด

โครงสร้างระบบต้องสามารถขยายไปสู่ระบบ Payroll, Attendance, Analytics และการบริหารหลายสาขาได้ในอนาคต โดยไม่ต้องเปลี่ยนสถาปัตยกรรมหลัก

# Authentication

- ใช้ Google OAuth เท่านั้น
- ไม่ต้องมีระบบสมัครสมาชิก
- Admin เป็นผู้กำหนดสิทธิ์ผู้ใช้งาน

## เมื่อ Login ครั้งแรก

- ตรวจสอบอีเมล
- ถ้ายังไม่มีในระบบ ให้สร้างบัญชีอัตโนมัติ
- กำหนด Role เป็น Assistant
- Admin สามารถเปลี่ยน Role ภายหลังได้

## Role

- Super Admin
- Admin
- Assistant

# User Workflow

1. Login ด้วย Google
2. เข้าสู่ Dashboard
3. ดูรายได้
   - วันนี้
   - สัปดาห์นี้
   - เดือนนี้
4. กดเพิ่มรายการงาน
5. เลือกวันที่
6. เลือกหมวด
7. เลือกรายการหัตถการ
8. กรอกจำนวน (Quantity)
   - Quantity สามารถเป็นตัวเลขใดก็ได้ เช่น จำนวนเคส, CC, Unit, นาที, ชั่วโมง
   - ระบบจะคำนวณอัตโนมัติ
9. กดบันทึก
10. ระบบอัปเดต Dashboard ทันที
11. สามารถกลับมาแก้ไขหรือลบรายการได้ตลอด

---

# Admin Workflow

Admin สามารถ

- ดูข้อมูลของผู้ใช้ทุกคน
- ดู Dashboard ภาพรวม
- ดูรายได้รวมของทีม
- ดูข้อมูลรายวัน รายสัปดาห์ รายเดือน
- แก้ไขข้อมูลของผู้ใช้
- เพิ่ม/ลบ/แก้ไข Procedure
- เพิ่ม/ลบ/แก้ไข Category
- เปลี่ยนอัตราค่ามือ (Rate)
- Export รายงาน
- ดูประวัติการแก้ไข (Audit Log)
- ปิดการใช้งานผู้ใช้
